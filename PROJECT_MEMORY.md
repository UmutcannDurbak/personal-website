## [1. PROJECT VISION & GOALS]
* **Core Concept:** A professional personal portfolio website to showcase achievements, skills, and personal growth for career opportunities.
* **Target Audience:** Recruiters, HR professionals, Potential Employers, Business Partners.
* **Success Criteria:** A visually stunning, responsive, and professional website that effectively communicates the user's value proposition and leaves a strong positive impression.

## [2. TECH STACK & CONSTRAINTS]
* **Language/Framework:** HTML5, CSS3 (Variables + Flex/Grid), Vanilla JavaScript.
* **Backend/DB:** Static hosting (GitHub Pages).
* **State Management:** DOM-based.
* **Key Packages:** None.
* **Constraints:** Mobile responsive, Dark Mode Default, Glassmorphism aesthetic.

## [3. ARCHITECTURE & PATTERNS]
* **Design Pattern:** Static Single Page Application (SPA) feel.
* **Folder Structure:**
    * `/`: Root HTML/JS/CSS
    * `/logos`: Brand assets
    * `/locales`: JSON translation files
* **Naming Conventions:** Kebab-case for classes, CamelCase for JS variables.

## [4. ACTIVE RULES (The "Laws")]
*(Yapay zekanın asla çiğnememesi gereken kurallar)*
1.  Use semantic HTML5.
2.  Ensure fully responsive design (Mobile First).
3.  Prioritize visual aesthetics and "wow" factor (Modern UI/UX).
